# Key
Programming Language written in Typescript.

# File Extension
* *.ky

# Keywords
* `let`
* `const`
* *MORE TO COME*

# Operators
* Assign `<-`
* Binary Operators
  * Addition `+`
  * Subtraction `-`
  * Multiplication `*`
  * Division `/`
  * Modulus `%`
* Grouping [Example](https://github.com/Winter-r/Key/blob/main/Examples/test.ky)
